# {PhotoFlow} 📸

Веб-приложение для публикации фотографий с умным поиском по тегам, заголовкам и описаниям. Проект позволяет пользователям создавать персональные галереи, делиться снимками и обсуждать их.

##  Основные возможности

- **Управление контентом**: Загрузка фотографий, редактирование описаний и удаление.
- **Система тегов**: Присваивание меток к фото для удобной категоризации.
- **Умный поиск**: Поиск по названию, описанию или нескольким тегам одновременно (с поддержкой автодополнения Select2).
- **Личный кабинет**: Раздел «Мои фотографии» для управления собственным контентом.
- **Социальное взаимодействие**: Система комментариев к каждой фотографии.
- **Безопасность**: Полноценная авторизация и регистрация пользователей (Flask-Login).
- **Справочный раздел**: Встроенная страница помощи с инструкциями по использованию.

##  Стек технологий

- **Backend:** Python, Flask, Flask-Login
- **Database:** SQLAlchemy (ORM), SQLite
- **Forms:** Flask-WTF

##  Инструкция по запуску

Для работы приложения необходим Python 3.7+.

1. **Клонируйте репозиторий:**
   ```bash
   git clone [https://github.com/kepchupchik/web-project.git](https://github.com/kepchupchik/web-project.git)
   cd web-project
2. **Настройте виртуальное окружение:**

Bash
python -m venv venv
# Для Windows:
venv\Scripts\activate
# Для macOS/Linux:
source venv/bin/activate
Установите необходимые библиотеки:

Bash
pip install -r download.txt
Запустите приложение:

Bash
python main.py
Приложение будет доступно по адресу http://127.0.0.1:5000.

📂 Структура проекта
main.py — основной файл приложения, маршруты и логика.

data/ — модели базы данных (User, Photo, Tag, Comment).

forms/ — формы для регистрации, входа и добавления контента.

templates/ — HTML-шаблоны на базе Jinja2.

static/ — стили, скрипты и загруженные изображения.

Проект разработан в качестве учебной работы по веб-программированию на Python.
